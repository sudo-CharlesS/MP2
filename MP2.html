Le Projet

But du projet

L’objectif était de réaliser un logiciel permettant de crypter ou de décrypter un texte par substitution mono-alphabétique, avec le langage de programmation Python.

En cryptographie, le chiffrement par décalage, qui est connu aussi comme le cryptage de César, est une méthode de cryptage très simple, utilisée par Jules César dans ses correspondances secrètes.

Le principe du cryptage par substitution mono-alphabétique est de décaler le rang des lettres dans l’alphabet, selon une clé donnée, ce qui rend le texte indéchiffrable en l’état. Par exemple avec une clé de 3, la lettre A devient la lettre D, et ainsi de suite. Il y a 25 clés possibles. En effet, il y a 26 lettres dans l’alphabet, mais une clé de 26 reviendrait à une clé de 0, car on ferait le tour complet de l’alphabet. De mème, si on utilise une clé supérieur à 26, cela reviendrait à une clé entre 1 et 25. Et enfin, si on utilise une clé négative, on parcourt l’alphabet dans le sens inverse.

Pour tous les caractères autres que des lettres majuscules, comme les espaces, les chiffres ou la ponctuation, ils restent inchangés dans le texte crypté.

Le programme que nous avons réalisé comporte un programme principal accompagné de trois fonctions : une fonction de cryptage avec une clé donnée, une fonction décryptage avec une clé donnée également, et enfin une fonction de décryptage sans clé, qui utilise également la deuxième fonction.

Ce logiciel s’utilise en ligne de commande, mais il pourrait également être exploité par un autre programme Python.

Pour réaliser notre programme nous avons utilisé l'IDE Spyder.

Le but de ce projet était d’exploiter les connaissances apprises précédemment, comme les fonctions, les conditions, les boucles, ou encore les différents types de variables. Egalement, comme ce projet était à réaliser par groupes de deux, nous avons put travailler en équipe et améliorer nos méthodes de communications. Le projet

Lorsque l'on éxcute le programme, nous avons 3 options :

  1. Cryptage avec clé
  2. Décryptage avec clé
  3. Décryptage sans clé
  4. Fin du programme

La partie du programme que j’ai le plus appréciée est la fonction décryptage sans clé, car c’est celle qui nous a demandé le plus de réflexion. En effet, cette fonction devait permettre à l’utilisateur de décrypter une chaîne de caractère sans la clé de décryptage. Pour cela, la fonction décryptage sans clé utilise la fonction décryptage avec clé afin de tester successivement les 25 clés possibles, et demande à l’utilisateur à chaque clé testé s’il visualise le texte en clair, et ce jusqu’à ce que le texte soit décrypté ou que les 25 clé aient étés testés. Dans ce cas le programme renverra un message d’erreur. 

def decryptageSansCle(text:str)->str:                       #définit fonction decryptageSansCle avec text comme paramètre
                """
                Décrypte une chaine sans clé    
                \n Pré : texte à décrypter -> text:str
                \n Post : texte décrypté -> textDecrypte:str ou message erreur décryptage -> str 
                """
            
                assert type(text)==str, "Le texte doit être une chaine de caractères"    #affiche si text n'est pas une chaine de caractères
                i=0                                                     #définit variable i = 0
                while i <= 26:                                          #répète les instructions ci-dessous tant que i est inférieur ou égal à 26
                    print("\n Tentative de décryptage avec clé de {} : {}".format(i, decryptage(i, text)))     #affiche le message précèdant avec la variable i et le résultat de la fonction decryptage avec les paramètres i et text
                    textClair=input("Si vous visualisez le texte en clair tapez F (majuscule ou minuscule), sinon tapez entrée : ")         #assigne à la variable textClair le choix de l'utilisateur
                    if textClair=='f' or textClair=='F':                #si textClair = lettre f ou si textClair = lettre F
                        return decryptage(i, text)                      #retourne le résultat de la fonction decryptage avec les paramètres i et text
                    else:                                               #sinon
                        i+=1                                            #ajoute 1 à i
                return "Désolé, vous avez essayé toutes les valeurs de clé, le texte a été crypté selon une méthode non mono-alphabétique"     #retourne le texte précédant

La réalisation

Les difficultés :

Nous n’avons pas particulièrement rencontrés de difficultés, néanmoins ce projet nous aura demandé quelques réflexions quant à l’utilisation de clés négatives, ou de la fonction décryptage sans clé par exemple.

Les perspectives :

Comme perspectives nous aurions pu apporter au programme une gestion des lettres minuscules ou encore ajouter une interface graphique afin de simplifier l’utilisation de logiciel.
